# 05 用户使用手册

> **项目名称**：智慧物业管理系统
> **适用对象**：物业管理人员（超级管理员、物业经理、收费员、维修员、业主）
> **文档版本**：V1.0
> **编写人**：王茂吉

---

## 一、系统概述

智慧物业管理系统是一套面向住宅小区物业的 B 端管理平台，帮助物业人员高效完成小区基础信息维护、报修工单处理、费用收取、车位与设备管理、数据统计等工作。

本手册面向系统的实际使用者，指导各角色人员完成日常操作。

---

## 二、登录与注销

### 2.1 访问系统
1. 打开浏览器（推荐 Chrome 或 Edge 最新版）。
2. 地址栏输入系统访问地址（开发环境：`http://localhost:5173`）。
3. 进入登录页面。

### 2.2 登录
1. 在登录页输入 **账号** 和 **密码**。
2. 点击「登录」按钮。
3. 登录成功后自动跳转至数据看板首页。

| 角色 | 默认账号 | 默认密码 |
|------|---------|---------|
| 超级管理员 | admin | admin123 |
| 物业经理 | manager01 | 123456 |
| 收费员 | toll01 | 123456 |
| 维修员 | repair01 | 123456 |
| 业主 | owner01 | 123456 |

> ⚠️ **安全提示**：首次登录后请尽快在「个人中心」修改默认密码。

### 2.3 修改密码
1. 点击页面右上角用户头像，选择「修改密码」。
2. 输入旧密码、新密码、确认新密码。
3. 点击「确定」完成修改，修改后需重新登录。

### 2.4 注销
点击页面右上角用户头像，选择「退出登录」，返回登录页。

## 三、界面布局说明

登录后进入系统主界面，布局分为三个区域：

```
┌──────────────────────────────────────────────┐
│  顶部导航（折叠按钮 | 面包屑 | 用户下拉）        │
├────────────┬─────────────────────────────────┤
│            │                                 │
│  侧边栏    │         主内容区                 │
│  菜单      │    （当前操作的页面内容）          │
│            │                                 │
│            │                                 │
└────────────┴─────────────────────────────────┘
```

- **侧边栏菜单**：根据当前登录用户的权限动态显示，不同角色看到的菜单不同。
- **顶部导航**：左侧为面包屑（显示当前页面路径），右侧为用户操作下拉。
- **主内容区**：展示当前操作的功能页面。

## 四、各角色操作指南

### 4.1 超级管理员

超级管理员拥有系统全部权限，主要负责系统配置与用户权限管理。

#### 用户管理
1. 进入「系统管理 → 用户管理」。
2. **新增用户**：点击「新增」按钮，填写用户名、姓名、手机、邮箱、所属小区、用户类型，点击「确定」。
3. **分配角色**：点击用户行的「分配角色」，勾选对应角色，点击「确定」。
4. **重置密码**：点击「重置密码」，用户密码恢复为默认值。
5. **启用/禁用**：点击状态开关切换账号启用状态。

#### 角色与权限管理
1. 进入「系统管理 → 角色管理」。
2. **新增角色**：填写角色编码（如 `TOLL_COLLECTOR`）、角色名称。
3. **配置权限**：点击「分配权限」，在权限树中勾选该角色可用的菜单与按钮，点击「确定」。

### 4.2 物业经理

物业经理负责本小区全部业务运营。

#### 基础数据维护
1. **小区管理**：「基础数据 → 小区」，维护小区基本信息。
2. **楼栋管理**：「基础数据 → 楼栋」，新增楼栋时选择所属小区。
3. **房屋管理**：「基础数据 → 房屋」，新增房屋时选择所属楼栋，填写房号、户型、面积、状态。
4. **业主管理**：「基础数据 → 业主」，登记业主信息（姓名、电话、身份证、车牌）。
5. **业主-房屋绑定**：在业主详情中点击「关联房屋」，选择房屋并标记关系（户主/配偶/子女/租客）。

#### 报修工单处理
1. 进入「业务管理 → 报修工单」。
2. **派单**：对待派单工单点击「派单」，选择维修员，点击「确定」。
3. **查看进度**：在工单列表中查看工单状态（已派单/处理中/已完成）。
4. **关闭工单**：业主评价后，工单自动关闭归档。

### 4.3 收费员

收费员负责账单生成与费用收取。

#### 账单生成
1. 进入「业务管理 → 账单管理」。
2. 点击「批量生成」，选择收费项目、账期、目标房屋范围，点击「确定」，系统自动生成应收账单。

#### 缴费登记
1. 在账单列表找到待缴费账单（状态：未缴/部分缴纳）。
2. 点击「缴费」，输入实缴金额、选择支付方式（现金/微信/支付宝/银行转账/POS），点击「确定」。
3. 全额缴费后账单状态变为「已缴清」，部分缴费变为「部分缴纳」。

#### 账单查询与导出
1. 按状态、账期、房屋筛选账单。
2. 点击「导出」可下载 Excel 对账表。

### 4.4 维修员

维修员负责接收并处理报修工单，执行设备巡检。

#### 处理工单
1. 进入「业务管理 → 报修工单」。
2. 列表仅显示分配给自己的工单。
3. **接单**：对待处理工单点击「接单」，状态变为「处理中」。
4. **完工反馈**：维修完成后点击「完工」，填写处理说明，状态变为「已完成」。

#### 设备巡检
1. 进入「业务管理 → 设备管理」。
2. **巡检登记**：对待巡检设备点击「巡检」，选择结果（正常/异常），异常时填写描述，点击「确定」。
3. **到期预警**：临近巡检日期的设备会在列表中高亮提示。

### 4.5 业主

业主可查看本人房屋与账单信息，提交报修。

#### 提交报修
1. 进入「业务管理 → 报修工单」。
2. 点击「报修」，选择报修类型（水电/土建/门窗/公共设施/其他），填写问题描述，提交工单。
3. 在列表中查看工单处理进度。
4. 工单完成后可对服务进行评分评价。

## 五、数据统计使用

### 5.1 综合数据看板
登录后默认进入看板首页，展示：
- **KPI 卡片**：房屋总数、入住率、本月收费、待处理工单数
- **收费趋势图**：近 6 个月实收趋势
- **工单状态分布**：各状态工单占比
- **设备状态**：在线/离线/故障统计
- **待办提醒**：待派单工单、到期账单、设备巡检到期

### 5.2 统计报表
- **收费统计**：「数据统计 → 收费统计」，查看应收/实收/欠费汇总与趋势。
- **工单统计**：「数据统计 → 工单统计」，查看工单按状态、按处理人分布。
- **车位使用率**：「数据统计 → 车位统计」，查看车位占用情况。
- **设备状态**：「数据统计 → 设备统计」，查看设备运行状态分布。

## 六、常见问题解答（FAQ）

**Q1：登录提示"账号已禁用"怎么办？**
A：账号被管理员禁用，请联系超级管理员在「用户管理」中启用账号。

**Q2：忘记密码怎么办？**
A：联系超级管理员在「用户管理」中点击「重置密码」，密码将恢复为默认值。

**Q3：为什么我看不到某个菜单？**
A：菜单显示受权限控制。请联系管理员确认你的角色是否分配了对应权限。

**Q4：工单派单后维修员看不到工单？**
A：请确认派单时选择的维修员账号是否正确，且该维修员已登录并具有工单处理权限。

**Q5：账单缴费后状态没有变为"已缴清"？**
A：账单状态根据「应收金额」与「已缴金额」自动计算。若为部分缴费，状态为「部分缴纳」；只有累计实缴 ≥ 应收时才变为「已缴清」。

**Q6：导出的 Excel 打不开？**
A：请确保使用 Microsoft Excel 或 WPS 打开；若仍无法打开，检查浏览器是否拦截了下载。

**Q7：页面数据不刷新？**
A：尝试按 F5 刷新页面，或清除浏览器缓存后重新登录。

**Q8：操作时提示"无权限"？**
A：当前账号无该操作权限。按钮级权限通过 `v-permission` 控制，若可见按钮提示无权限，请联系管理员检查权限配置。

## 七、操作注意事项

1. **数据安全**：删除操作不可恢复（逻辑删除），请确认后再执行。
2. **并发操作**：多人同时编辑同一数据时，以最后保存的为准。
3. **浏览器**：推荐使用 Chrome 或 Edge 最新版，不推荐 IE 浏览器。
4. **退出习惯**：离开工位时请注销登录，防止账号被盗用。
5. **数据录入**：身份证、手机号等关键字段请确保准确，系统会进行唯一性校验。
