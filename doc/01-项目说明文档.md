# 01 - 项目说明文档

## 1. 项目简介

**小区物业管理系统**是一套面向住宅小区物业管理业务场景的 B 端管理系统，覆盖物业日常运营中的「人、房、车、钱、事、设备」六大维度，目标是把小区的基础信息、收费、报修工单、车位、安防消防设备等业务从纸质/Excel 升级为线上化、可统计、可追溯的信息化管理。

系统采用 **前后端分离** 架构：

- 后端：SpringBoot 2.7 + MyBatis，提供 RESTful 接口
- 前端：Vue3 + Vite + Element Plus，单页应用（SPA）
- 数据库：MySQL 8

## 3. 角色定义

系统采用 RBAC 权限模型，预置以下角色：

| 角色 | 编码 | 说明 |
| ---- | ---- | ---- |
| 超级管理员 | `SUPER_ADMIN` | 拥有全部权限，不可删除 |
| 物业经理 | `PROPERTY_MANAGER` | 管理本小区全部业务、可看全部报表 |
| 收费员 | `TOLL_COLLECTOR` | 费用管理、收费、账单 |
| 维修员 | `REPAIRMAN` | 接单、处理工单、填写处理结果 |
| 业主 | `OWNER` | 仅查看本人房屋、账单、报修、车牌 |

## 4. 功能模块（来自功能规划表）

整体分为 **5 大一级模块**：

```
小区物业管理系统
├── ① 用户认证模块
│   ├── 登录
│   ├── 注销
│   └── Token 全局鉴权
├── ② 系统管理模块
│   ├── 用户管理
│   ├── 角色管理
│   └── 权限管理
├── ③ 基础数据模块
│   ├── 小区管理
│   ├── 楼栋管理
│   ├── 房屋管理
│   └── 业主信息管理
├── ④ 业务功能模块
│   ├── 工单管理
│   ├── 费用管理
│   ├── 车位管理
│   ├── 通闸设备管理
│   ├── 消防设备管理
│   └── 安防设备管理
└── ⑤ 数据统计模块
    ├── 仪表盘
    ├── 收费报表
    ├── 工单报表
    └── 报表导出
```

### 4.1 用户认证模块

- **登录**：账号 + 密码登录，密码经 BCrypt 加密存储；登录成功返回 JWT Token。
- **注销**：前端清除 Token，后端将 Token 加入黑名单（可选）。
- **Token 全局鉴权**：除 `/api/auth/login` 等白名单接口外，所有接口必须携带 `Authorization: Bearer <token>`；后端用拦截器统一校验，并把当前登录用户放入 ThreadLocal。

### 4.2 系统管理模块（RBAC）

- **用户管理**：物业员工账号的增删改查、分配角色、重置密码、启用/禁用。
- **角色管理**：角色的增删改查、给角色分配权限（菜单 + 按钮）。
- **权限管理**：菜单权限、按钮权限的维护；前端动态路由 + 按钮级控制。

### 4.3 基础数据模块

- **小区管理**：小区基本信息（名称、地址、开发商、建成年份、绿化率等）。
- **楼栋管理**：楼栋归属小区，楼栋层数、单元数、电梯数。
- **房屋管理**：房屋归属楼栋，房号、面积、户型、是否入住、绑定业主。
- **业主信息管理**：业主姓名、电话、身份证、车牌、入住时间、状态。

> 房屋与业主为多对多关系（一套房可有多名业主，一名业主可拥有多套房）。

### 4.4 业务功能模块

| 子模块 | 关键能力 |
| ---- | ---- |
| 工单管理 | 业主报修 / 物业派单 / 维修员接单处理 / 业主评价；工单状态流转 |
| 费用管理 | 物业费 / 水费 / 电费 / 车位费账单生成、收费、欠费统计 |
| 车位管理 | 车位（地上/地下、长租/出售）登记、绑定车牌、缴费到期 |
| 通闸设备 | 门禁闸机登记、在线状态、通行记录 |
| 消防设备 | 灭火器/消防栓登记、巡检记录、到期提醒 |
| 安防设备 | 摄像头登记、在线状态、录像时段 |

### 4.5 数据统计模块

- **仪表盘**：总户数、总人数、本月收费、待处理工单、设备在线率等关键指标 + 趋势图。
- **收费报表**：按小区 / 楼栋 / 费用类型 / 时间段统计应收、实收、欠费。
- **工单报表**：按状态、类型、维修员统计工单数量与平均处理时长。
- **报表导出**：支持 Excel 导出。

## 6. 术语表

| 术语 | 含义 |
| ---- | ---- |
| RBAC | Role-Based Access Control，基于角色的访问控制 |
| JWT | JSON Web Token，无状态鉴权令牌 |
| 工单 | 业主报修 / 物业巡查产生的一条业务记录 |
| 账单 | 某房屋某周期某费用类型的应收记录 |
| 欠费 | 账单已过缴费截止日但未缴清 |
