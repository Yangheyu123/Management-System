## 阅读建议

1. **了解项目**：先读 [01 需求分析](./01_需求分析.md)
2. **对接前后端**：读 [03 API 文档](./03_API文档.md)
3. **建库**：参考 [02 数据库设计](./02_数据库设计.md)，执行 `db/schema.sql` + `db/data.sql`
4. **部署运行**：照 [05 用户手册](./05_用户手册.md) 第一部分操作
5. **快速体验**：默认账号 `admin` / `admin123`

## 接口约定速览

- Base URL：`/api`
- 鉴权：请求头 `Authorization: Bearer <token>`
- 统一返回 `Result<T>`：`{ code, message, data }`
- 时间格式：`yyyy-MM-dd HH:mm:ss`
